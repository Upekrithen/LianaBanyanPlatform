# 📚 Glossary

## Cooperative-class terminology used throughout Cephas.

---

**83.3% creator-keep** — The platform constant: on every transaction, the creator (worker / artist / service provider) keeps 83.3%. Result of the Cost+20% margin math.

**SSPL v1** — Server Side Public License v1.0. The cooperative's primary software license. Source available; if you run modified SSPL software as a service, you must open-source the entire service stack. Distinct from AGPL — SSPL has broader network-service copyleft.

**Aircraft Carrier doctrine** — The substrate is a continuously-operating platform from which Bishop launches waves without bringing down the whole ship for re-staging.

**Ambassador Position** — Political Expedition Council Precept. Any member can stand as Ambassador for a coalition. Self-funded + member-matched + half-redistributed-equally to other coalitions.

**Banyan Almanac** — The cooperative's periodic publication documenting platform-state, empirical receipts, and milestone retrospectives. Issue 003 covers BP038 + BP039.

**Bishop** — One of four specialized AI instance-roles in the Conducted AI Substrate Orchestra. Bishop = foreman; architectural decisions; canon ratification. Claude Opus 4.7 / Sonnet 4.6 K-lineage.

**Blood Rule** — A Trinity Rule of Blood-class. Structurally protected; no instance violates regardless of instruction. Currently 7 Blood-class rules in the Trinity Rules SEPTENDECIM 17.

**Body-Cam-Survivability** — The vetting standard for Council members and Crown candidates. Public record must stand under full transparency. Not perfect — honest about it.

**BP** — Bishop Pod numbering. BP039 is the current Bishop session as of this canon. Sessions are numbered sequentially.

**Bounty Poster** — "I'm Just a Bill" pilot artifact. Plain-language explanation of a real U.S. bill, Council-vetted via the Shirley Temple Truth Score.

**Brick Wall Test** — Founder-direct discipline that asks Bishop to produce N× more output before context exhausts, demonstrating substrate throughput at the limit.

**CAI** — Conducted AI Substrate Orchestra. The cooperative's engineering architecture. Multi-instance AI specialization conducted by human operator through substrate-canon protocol. CAI-4.5 = 31,833× efficiency multiplier (empirically validated BP038/BP039).

**Cephas** — The cooperative's consumer-facing content surface (Hugo static-site). Cephas = "rock" (foundational).

**Coffee** — The canonical session-handoff artifact. Each BP closes with a Coffee that the next session-Bishop reads on open.

**Cooperative Compact** — The five constants: 83.3% creator-keep + Cost+20% margin + $5/year membership + three-currency loop + bylaw-codified Structural Bylaws.

**Cost + 20%** — The platform margin constant. Fixed by structure (bylaws); cannot be raised.

**Council** — The 30-member Political Expedition Council (Initiative #15). Civic-information vetting body.

**Credits / Marks / Joules** — The three currencies of the cooperative's internal economy. Substitution-only; never convert to fiat.

**Crown** — The designated leader of an Initiative. Receives a Crown letter on Cephas. Operates the Initiative within the Cooperative Compact.

**Designed To Pass On** — Foundational paper authored BP038. Dissolution-day-zero design discipline.

**Drekaskip** — The Wave Generator. Cooperative-AI parallel-fan-out infrastructure. Each wave fires multiple SEGs that produce independent output then synthesize.

**Eblet** — A canonical substrate artifact. Markdown file with frontmatter; stored in `.claude/state/eblets/CANON/`. Audit-grade canon record.

**Excalibur Saga** — A saga-class wave fire. Sequential multi-wave fire under unified saga_id. BP039 fired 4 such sagas.

**Founder** — Jonathan R. Jones. The platform's architect and primary human operator. Roles: AI Tuner + The QueTuner.

**Galveston Legal Framework** — The jurisdictional-anchor for first-deployment of the LBCCL Healthcare licensing structure (Initiative #6 Tatiana Schlossberg Health Accords).

**Helena Pedagogy** — Cooperative teaching discipline: answer-first, then question, then spaced repetition. No gotchas.

**Hugo** — Static site generator. The cooperative uses Hugo for Cephas content.

**"I'm Just a Bill"** — The Political Expedition Council's pilot project. Bounty Posters for real legislation in plain language. Named for Schoolhouse Rock 1975 cartoon.

**Initiative** — One of the 16 Sweet Sixteen operational scopes. Each has a Crown (or 30-Council for #15, or TRIBUTE-class for #6).

**Joules** — One of the three cooperative currencies. Productivity-value.

**Knight** — One of four AI instance-roles. Knight = executor; code-class work; substrate writes. Cursor with GPT 5.5 / Sonnet 4.6.

**LBCCL** — Liana Banyan Cooperative Commons License. Cooperative-class licensing framework (variants for Healthcare, Learner-Benefit, etc.).

**Listen Before Talk** — Political Expedition Precept. In any cooperative deliberation, members listen to all positions before any member speaks.

**Marks** — One of the three cooperative currencies. Recognition-value.

**MEMORY.md** — Bishop's mnemonic index. Mnemosyne discipline (≤200 lines).

**Mnemosyne** — MEMORY-compaction discipline. Index pointing to detail-Eblets rather than holding detail itself.

**NO-FIAT-CONVERSION / SUBSTITUTION-ONLY** — Blood Rule. Cooperative currencies never convert to fiat.

**Not Left. Not Right. Forward.** — Political Expedition Council tagline. Forward ≠ centrist. Forward = build cooperative-class infrastructure that makes the question moot.

**Pawn** — One of four AI instance-roles. Pawn = verifier. Perplexity sonar-reasoning-pro.

**Pedestal Vote** — Canonical record of a Crown-candidate response (acceptance / refusal / non-response). Public canon.

**Pheromone** — Cross-reference index across substrate canon. Updated on every librarian rebuild.

**R16** — Blood Rule: NO-API-KEY-EXPOSURE. Never surface key values in any UI / tool output / artifact.

**R17** — Blood Rule: SHOW-RESULTS. Cross-probe independent observers before declaring async dispatch "complete."

**Rook** — One of four AI instance-roles. Rook = researcher. Gemini 3.1 Pro flagship.

**Saga** — A sequential multi-wave fire under unified saga_id. G-Saga gates verify saga-level health.

**SEG** — Saga Element Generator. One worker-instance within a wave. Each wave fires 8 SEGs + 1 synthesis SEG.

**Shirley Temple Truth Score** — Council vetting standard. *If Shirley Temple would say it on TV, it passes.*

**Substitution** — The mechanism by which cooperative currencies are used at participating providers. Never conversion to fiat.

**Sweet Sixteen** — The 16 Initiatives. Bylaw-codified count.

**Trinity Rules** — The cooperative's structural-discipline canon. Currently 17 rules (SEPTENDECIM). Seven are Blood-class.

**TRIBUTE class** — Crown class for Initiatives named after deceased namesakes. Initiative #6 (Tatiana Schlossberg Health Accords) is the only current TRIBUTE-class Initiative.

**Wave** — A single Drekaskip parallel-fan-out fire. 8 SEGs + 1 synthesis SEG per wave.

**WonderBoy Succession** — Cooperative hand-off architecture. *The Natural* reference. Founder direct: *"I'm down for that."*

**X=16** — Council escalation threshold. When 16 of 30 Council members reach accepted/active status, the Congress / SCOTUS / POTUS invitation flow triggers.
