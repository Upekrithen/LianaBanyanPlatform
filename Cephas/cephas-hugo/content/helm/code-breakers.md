# Code Breakers Corps Guild

The Code Breakers Corps is a **cross-cutting Guild** within the LB Frame federation. Members may enroll without leaving their primary Tribe, Family, or Guild affiliations — the Guild composes alongside all existing affiliations.

## What the Guild Does

Members solve **Standing Security Bounties** — a permanent set of security-class puzzles spanning cryptographic ciphers, substrate-routing discovery, canon Eblet audits, Scribe edge-cases, and brand-fence identification. Every solved puzzle earns:

1. **Marks payout** — Tier 5 Security class, 3.0× multiplier
2. **IP Ledger stamp** — a permanent row in the Public IP Ledger crediting the solver's name
3. **First-solve marker** — the first member to solve each bounty receives the canonical artifact attribution permanently

*LB membership: $5/year, identical for all members from day one through year fifty.*

## IP Ledger Integration

Each solve fires an `ip_ledger_stamp` row with:

- `stamp_class = code_breakers_solve`
- `first_solve_marker = true` for the first solver on each bounty
- Attribution routes to the **10% individual Pedestals IP allocation slice** per Public IP Ledger canon

Members may also author a **Pedestal contribution decree** on the bounty's canonical artifact — the original bounty and the solver's Pedestal compose alongside (immutable + extending decree class per Mordecai-Esther Pedestal Forum canon).
