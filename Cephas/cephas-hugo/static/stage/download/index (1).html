Earn Marks for this Tower

Cooperative participation earns Marks — the network’s contingent accounts payable. Every contribution below improves every other member’s accuracy.

+2

Report a successful install. (Send to cooperative Discord or open a GitHub issue tagged install-report.)

+3

Capture a Discord or Reddit thread about this version → MnemosyneC Search → 📚 Save → eblet minted.

+10

Report a bug that triggers a version demotion. Truth earns the most.

+25

Stone-Tablet-truth bonus. Rare. A verified deep-canon contribution that becomes load-bearing canon.

▶  How do versions get promoted or demoted?
🟡 LATEST → 🟢 STABLE: 10 verified install reports + 30 days zero-issue + 3 verified eblets minted on this version.
🔵 HISTORICAL → 🔴 DEPRECATED: critical bug confirmed in this version that cannot be backport-fixed.
🟢 STABLE Demotion trigger: any P0 regression discovered. Stability is earned; it is not permanent.

Trust scores are the substrate’s living ledger for each version. Hover any score to see the formula breakdown.

Catacombs Trust Accumulator

Each version’s trust score compounds from three streams and one penalty:

  • Verified install reports × 2 — a human ran it and told us.
  • Verified eblets minted on this version × 1 — cooperative knowledge stamped to this rung.
  • Zero-issue days since release × 1 — silence is evidence. Time compounds trust.
  • Open issue reports × −10 — a known problem cannot hide inside a high score.

The score is build-time static. Dynamic scoring (real-time Marks feed) ships in a later version of the Tower.
