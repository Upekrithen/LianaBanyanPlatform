What Is MnemosyneC?#

MnemosyneC is a local-first AI memory substrate. It gives any AI — ChatGPT, Claude, Gemini, Cursor, any model — a permanent, private, verifiable memory that lives on your machine.

The name comes from Mnemosyne (neh-MOZ-uh-nee), the Greek goddess of memory and the mother of the Muses. Dr. MnemosyneC inherited her memory.
