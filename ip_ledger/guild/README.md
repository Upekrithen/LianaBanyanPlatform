# Substrate Developer's Guild

## What This Is

The Substrate Developer's Guild is **Spinout #2** -- a dedicated cooperative arm for
developers who build on the MnemosyneC substrate.

This is distinct from Spinout #1 (CAI Bonfire), which is the cooperative AI contribution
and training spinout. The Guild is for builders who integrate, extend, and deploy
substrate-native applications using the LB platform primitives.

## Build on the Substrate

The MnemosyneC substrate provides:
- Pheromone signaling (stigmergic discovery index)
- Pearl DAG (content-addressable knowledge fragments)
- Soccerball (session DAG)
- IP Ledger (Ed25519-stamped provenance chain)
- Fleet broadcast + peer mesh (real-time cooperative networking)
- Keys and Engines (local-first AI inference layer)

Developers use these primitives to build cooperative applications across all 16 Sweet Sixteen
initiatives and beyond.

## Economic Model

- Platform charges **Cost + 20%** on substrate usage (compute, storage, relay bandwidth)
- Creators and workers keep **83.3%** of revenue generated via their Guild-built applications
  (on a $500 transaction, the creator receives $416.67)
- Membership: **$5/year** flat -- no tiers

## Marks Earn (Developer Track)

See `marks_earn_rules.md` for the full table. Key rates:

| Activity | Marks |
|---|---|
| Work completion (via work pool) | 10% rate on allocated work |
| Peer code review (verified) | 0.05 Marks per review |
| Bounty completion | Proportional to bounty size |
| Documentation contribution | 0.05 Marks per accepted doc |

All rates above are Founder-ratified. No new rates without Founder sign-off.

## Not Securities

Marks are cooperative participation units, NOT financial instruments.
Never use: equity, shares, dividends, ROI, invest.
Use: participation, allocation, contribution, back.
