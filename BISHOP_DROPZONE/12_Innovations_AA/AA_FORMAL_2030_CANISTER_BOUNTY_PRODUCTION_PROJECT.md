## Innovation #2030 — Canister System as Bounty-Funded Separate Project
**Category:** Manufacturing / Project Management / Cooperative Economics
**Impact:** HIGH
**Crown Jewel Candidate:** YES

### Description

The Canister System (#2022) launches as a **separate, bounty-funded project** alongside the SlottedTop Kickstarter Campaign 1. Rather than hiring a traditional manufacturing team, the entire vertically integrated production pipeline — CAD design, 3D printing, materials sourcing, assembly, QA, packaging, fulfillment — is built through the Bounty-to-Partnership pipeline (#2029).

**Production Targets:**
- **5,000 Canister Systems** (kits: sleeves, caps, base, weights, screw press, heated barrel)
- **15,000 SlottedTops** (hex terrain adapter — Campaign 1 flagship product)

**Launch Timing:** Both funded through the SlottedTop Kickstarter. Campaign 1 funds the first SlottedTop batch AND seeds the Canister System bounty pool. The Canister System ships as a separate product once the bounty-recruited team completes the production pipeline.

### The Vertically Integrated Bounty Pipeline

Every stage of production is a bounty:

| Stage | Bounties | Skills Needed | Marks Range |
|-------|----------|---------------|-------------|
| **CAD Design** | Finalize canister dimensions, sleeve tolerances, hex-key geometry, screw press threading | Fusion 360 / SolidWorks / FreeCAD designers | 50-200 Marks/task |
| **3D Print Validation** | Print test canisters in SLA/SLS, verify twist-lock fit, measure tolerances | Makers with SLA/SLS printers | 25-100 Marks/task |
| **Materials Sourcing** | Find suppliers for ACME screws, steel pistons, heating bands, PID controllers, PTFE seals | Supply chain / procurement | 50-150 Marks/task |
| **Mold Design** | Design production molds for sleeves, caps, bases (injection or thermoforming) | Mold design engineers | 100-300 Marks/task |
| **Assembly Process** | Design assembly jigs, write assembly instructions, time studies | Manufacturing / industrial engineering | 75-200 Marks/task |
| **QA Protocol** | Write test procedures: pressure test, twist-lock test, seal test, heat test | Quality engineers | 75-200 Marks/task |
| **Packaging Design** | Kit packaging, instruction cards, branded materials | Graphic design / packaging | 50-150 Marks/task |
| **Fulfillment Setup** | Shipping logistics, Factory Node distribution, regional warehousing | Logistics / operations | 100-250 Marks/task |

Total bounty pool needed: ~5,000-15,000 Marks to build the full pipeline from scratch.

### Why Separate Project?

1. **Risk isolation:** SlottedTop is proven (simple adapter). Canister System is complex (multi-component kit). Separate projects = separate risk.
2. **Different timelines:** SlottedTops can ship in 8-12 weeks. Canister Systems need 16-24 weeks for the full bounty pipeline.
3. **Different audiences:** SlottedTop buyers are gamers. Canister System buyers are MAKERS. Overlap exists but messaging differs.
4. **Separate Pedestal/Project pages** on the platform — each gets its own Bridge, its own Captain, its own community.
5. **Portfolio signal:** Two simultaneous projects = more activity = more engagement = stronger cooperative story.
