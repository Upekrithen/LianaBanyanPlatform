# Pudding #93 — The HexIsle

*A game world where the economics are real and the hexagons are the point.*

---

Most games with economies are designed to extract money from players. Loot boxes, premium currencies, pay-to-win mechanics, battle passes — the game's economy exists to convert player engagement into platform revenue. The player's time is the raw material. The platform's profit is the product.

HexIsle inverts this. It is a game world built on hexagonal geography where the economic mechanics mirror the cooperative's real economics. The currencies in the game — Credits, Marks, Joules — are the same currencies used across the rest of the platform. Effort in the game generates value that is recognized outside the game.

The hex grid is not decorative. Hexagons tile a plane with equal adjacency — every hex has exactly six neighbors, all equidistant. This creates a geography where no position is inherently advantaged by shape. A hex in the center has the same number of neighbors as a hex on the edge of a cluster. The geometry enforces fairness at the structural level.

HexIsle has a Spanish-language sibling: Hexislo.com. This is not a translation layer bolted onto an English product. It is a parallel surface — same game mechanics, same economics, same hexagonal geography — served in Spanish from a dedicated domain. The cooperative's membership is not English-only, and its game world should not be either.

The game uses a FocusShell — a full-screen immersive layout stripped of the platform's standard navigation chrome. When you are in HexIsle, you are in HexIsle. The sidebar, the top bar, the notification badges — all gone. The game demands your full attention, and the interface respects that demand by removing distractions.

HexIsle is not the cooperative's main product. It is one of 23 domains. But it demonstrates a principle that applies to the entire platform: economic systems can be designed so that participation generates value for the participant, not just for the system operator. In HexIsle, this principle takes the form of a game. Everywhere else on the platform, it takes the form of a cooperative.
