# Mnemosyne v0.1.11 Release Notes

**Release date:** 2026-05-24

---

## What's New

### Dashboard-first launch (SAGA-1)
On startup, Mnemosyne now opens the Dashboard by default. The overlay ("Burst Mode") is now opt-in — access it via the tray icon or the corner menu.

### 3-option corner menu (SAGA-2)
The bottom-right corner pill now opens a 3-option menu:
- **Open Dashboard** — bring up the main interface
- **Burst Mode** — activate the transparent AI overlay
- **Fallback Mode** — minimal overlay for low-resource sessions

### Tray icon restored (SAGA-3)
The NotCents Đ glyph tray icon has been restored to the known-good version.

### setOpacity overlay-show fix (H.8 · commit bc9817d)
`show-overlay` IPC handler now calls `setOpacity(1.0)` before `showInactive()`.
This prevents the overlay from remaining invisible after OS compositing holds opacity at 0 from a prior `hide()`.

### GDPR notice (SAGA-10)
A dismissible GDPR notice now appears on mnemosynec.ai.

### Accessibility (SAGA-11)
9 interactive elements now have proper aria-labels.
