### Application (technical specification, ~1500-2500 words)
1. **Real-time per-call marginal cost surface** — every call to a vendor inference API displays its marginal cost AT the point of call, not at end-of-session aggregate
2. **Running session total** — always visible
3. **Comparison delta on escalation** — when the application chooses a more expensive model over a cheaper one, the cost delta is annotated WITH the reason for escalation
4. **Default to cheapest available** — application's default model selection is the lowest-priced model meeting a configurable task-class threshold; escalation requires explicit user override OR demonstrated insufficient capability via empirical evidence
5. **Itemized post-session receipt** — every call categorized by purpose + cost
6. **Cross-vendor aggregate** — Personal Subscription Steward composition: monthly spend across Claude / Cursor / Google / Perplexity / ChatGPT GPT-5.5 / other surfaced by vendor, by purpose, by user-defined category
