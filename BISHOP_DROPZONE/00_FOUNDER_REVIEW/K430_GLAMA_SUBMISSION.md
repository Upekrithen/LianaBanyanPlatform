### Add librarian-mcp — intent-aware canonical memory + provenance

Independently benchmarked MCP server delivering pre-curated canonical memory packets with intent routing and prose provenance drift detection. 86.1 percentage-point accuracy lift across 8 models × 4 vendors × 1,200 graded calls (Eyewitness Benchmark R10, April 2026). Inter-rater κ 0.883/0.850.
